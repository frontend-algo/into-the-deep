## 2. 진행 방식

- 정해진 갯수만큼 [문제](https://www.teamblind.com/post/New-Year-Gift---Curated-List-of-Top-75-LeetCode-Questions-to-Save-Your-Time-OaM1orEU)를 풀어오기 (순서는 마음대로)

- 2번 이상 빠지는 경우, 스터디 진행 중간에 탈퇴하는 경우에는 벌금 10만원 💸
